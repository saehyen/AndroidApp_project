# 카페 원두 지도 앱

## 앱 설명
    - 카카오 계정 연동 로그인
    - 카카오 맵 연동
    - 지도에 있는 카페들 목록을 보여준다.
    - 카페들이 사용하는 원두의 정보와 원두의 맛(신맛 / 탄맛 / 중간맛)을 확인할 수 있다.
    - 각 카페별 리뷰를 확인 할 수 있다. (로그인을 하지 않아도 확인 가능, 카페별 별점 기능)
    - 리뷰를 작성하려면 로그인을 해야한다.(리뷰를 추천할 수 있다, 최다 추천 리뷰는 베스트 리뷰 선정(보상 지급)
